Intro to CSS

♡ Cascading Style Sheets or CSS is used to format the layout of a webpage.
♡ Cascading means that a style applied to a parent element will also apply to all children elements within the parent
♡ There are 3 ways of applying CSS to a HTML document:
---> Inline - the style attribute inside HTML elemments.
---> Internal - the style element is in the head section.
---> External - a link element is used to link to an external CSS file.

Some CSS examples:

Link to External CSS

<link rel="stylesheet" href="styles.css">

CSS Border

The border attribute draws a border around text.
