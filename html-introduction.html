Introduction to HTML

What is HTML?

♡ Hyper Text Markup Language
♡ Structure of a Web page
♡ Not case sensitive

HTML Elements

♡ They are defined by a start tag, content and an end tag.
♡ Some have no content or end tag, which are called empty elements.

HTML Attributes

♡ Attributes give additional information about elements.
♡ All elements have attributes.
♡ Attributes are specified in the start tag.
