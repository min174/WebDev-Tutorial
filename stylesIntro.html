HTML Style Attributes

♡ The HTML style attribute has the following syntax:
╰┈➤ <tagname style = "property:value;">

♡ Change the background colour for whole page:
╰┈➤ <body style = "background-color:blue;">

♡ Change the background colour for seperate elements
╰┈➤ <p style = "background-color:blue;">hello!</p>

♡ Draw a coloured border around a text:
╰┈➤ <p style = "border:2px red;">hello!</p>

♡ Change the text colour:
╰┈➤ <p style = "color:pink;">hello!</p>

♡ Change the font:
╰┈➤ <p style = "font-family:roboto;">hello!</p>

♡ Change the text size:
╰┈➤ <p style = "font-size:300%">hello!</h1>
