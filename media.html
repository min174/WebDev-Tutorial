Link element

♡ The <a> tag defines a hyperlink. Its called an anchor.
♡ The href attribute specifies the destination of the link.
╰┈➤ <a href = "www.google.com">This is a link</a>

Image element

♡ The <img> tag is used to embed an image in an HTML page.
♡ Images are linked to web pages.
♡ The <img> tag creates a holding space for the referenced image.
♡ src could be either be an Absolute Url or a Relative URL.
♡ Absolute URL is a link to an external website.
╰┈➤<img src="https://www.w3schools.com/images/lamp.jpg" alt="Lamp" width="100" height="100">
♡ Relative URL is a link to an image that is hosted by our website.
╰┈➤ <img src="god.jpg" alt="chuus face">
